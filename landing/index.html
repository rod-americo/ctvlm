What it does

🧪

Dual-encoder concat

L2-normalised concatenation of Merlin (2048-d) + Pillar-0 (1152-d) into a 3200-d feature, linear probe to 220 findings.

📐

Calibrated probabilities

Per-finding Platt scaling + Youden-J operating points + universal min-floor. Contrast-phase-aware on 26 high-prevalence findings.

🩻

Auditable sentences

Every positive finding maps to a tool-call recipe → measurement-bearing Tier A template or organ-anchored Tier B generic. No LLM in the production path.

🎯

Grad-CAM by finding

Per-encoder Grad-CAM heatmaps via the concat probe's weight slots. NIfTI output, displayable in NiiVue side-by-side.

🌐

Web viewer + DICOM upload

Streamlit dashboard with file-server sidecar. Browse pre-cached demos or drop in your own DICOM series.

🐳

Production-ready Docker

Multi-stage build. Encoder weights auto-download on first launch with your HF token. Volume-mounted cache persists across restarts.

Quick start

Two steps after cloning the repo:

# 1. Give the container your HuggingFace token
cp .env.example .env
# → edit .env, set HF_TOKEN=hf_... after accepting Merlin + Pillar-0 gates

# 2. Launch the viewer
docker compose up viewer
# → first launch downloads ~1.5 GB of encoder weights (one-time)
# → open http://localhost:8501

Or use the Python API directly:

from src.agent import pipeline
report = pipeline.generate_report("AC421363f", skip_llm=True, contrast_phase="ce")
print(report.summary_text)

Getting the data

The repo bundles:

  • RATE labels — 25,494 studies × 230 binary finding columns
  • Source radiology reports — the original report text (55 MB CSV)
  • Trained probe checkpoint — concat probe with Platt + Youden-J + NC calibration
  • 10 demo studies — pre-cached encoder features for instant browsing

The CT volumes themselves are not bundled. Download separately:

Limitations

⚠️ Research use only. This is NOT FDA-cleared / NOT CE-marked. Do not use as a sole basis for clinical decisions. The output is intended as a draft / triage / quality-check input to a qualified radiologist.
  • Trained on a single-institution US tertiary-care cohort (Merlin); external validity unverified
  • ~99% of training data is contrast-enhanced — non-contrast performance is measurably degraded (F1 0.510 vs 0.529)
  • Abdomen-pelvis only — do not route chest CT through this pipeline
  • Adult cohort — pediatric performance untested
